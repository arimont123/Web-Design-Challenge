Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis I first pulled data from the OpenWeather API to assemble a dataset. The dataset includes weather information on over 500 cities across the world with varying distance from the equator.

After assembling the dataset I used Python and Matplotlib to plot various attributes of Weather vs. Latitude. Factors that were analyzed include temperature, cloudiness, wind speed, and humidity. This site provides the images and descriptions of each image that were produced from my analysis as well as the data that was used.
